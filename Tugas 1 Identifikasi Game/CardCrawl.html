Goal

Menghabiskan 52 kartu yang ada di tangan musuh dan mendapatkan koin sebanyak mungkin.

Player Action

  • Pemain dapat memilih 3 dari 4 kartu yang dikeluarkan musuh.
  • Pemain dapat menyimpan maksimal 3 kartu item.
  • Pemain dapat menyerang dungeon card/monster dengan item pedang.
  • Pemain dapat menahan serangan musuh dengan kartu shield.
  • Pemain dapat melakukan heal dengan item potion.

Progression of Play

Permainan dimulai dengan memilih class. Pemain akan menghadapi 52 kartu yang dipegang musuh dan harus memilih strategi untuk bertahan hidup dan menang.

Opposition

  • Hasil kocokan kartu
  • Damage dari Dungeon card / Monster
  • Penggunaan Item

Game State

Elemen yang menentukan status permainan termasuk HP pemain, HP monster, hasil kocokan kartu, dan urutan kartu yang dikeluarkan.

Game Space

Area permainan terdiri dari 4 slot untuk kocokan kartu, 2 area penggunaan item, dan 1 inventory untuk menyimpan item.

Tugas 1.2 - Identifikasi Ide 5 Game

Non-Digital

  • Dungeon & Dragon: Fitur dadu untuk menentukan hasil gerakan / tindakan, ada fitur PVE / PVP.
  • Vanguard: Upgrade/advance Class dalam kondisi tertentu (deck building).
  • Shadowverse: Ada beberapa kartu item / region.

Digital

  • Metaphor: ReFantazio: Fitur Class inheritance.
  • Fire Emblem: Map kotak seperti SRPG dan efek terrain.
  • Reverse 1999: Setiap level kartu / skill memiliki efek tambahan / buff efek sebelumnya.
